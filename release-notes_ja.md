# リリースノート

## 6.8.3

2021年12月31日リリース。


今回のリリースの大きなの変更点は、プロジェクトレベルでコラボレーションができるようになったことです。Exploratoryサーバーにプロジェクトをパブリッシュして、他のチームメートとプロジェクトを共有できるようになりました。

### 機能強化

* コラボレーション: プロジェクトをパブリッシュして、他のチームメートと共有できるようになりました。
* コラボレーション: パブリッシュされた他のチームメートによる変更を自身のプロジェクトに反映できるようになります。
* コラボレーション: パブリッシュされたプロジェクトはバージョン管理されているので、過去のバージョンに復元することができます。
* コラボレーション: チームとしてプロジェクトをパブリッシュできます。これにより、チームメンバーとのコラボレーションしやすくなります。


* データソース: Amazon Aurora: SSL CAファイルのためのパラメーターをサポートしました。
* データソース: Google Analytics: データをサンプルをするかどうか選択できるようになりました。
* データソース: Google Sheets: Google Sheetsのサーバー上にあるシートを検索できるようになりました。
* データソース: Google BigQuery: ページサイズに関するエラーメッセージを改善しました。


* サマリビュー: 表示する列/行において、最初または最後から何列/何行を選択できるようになりました。
* サマリビュー: 相関モード: アルファベット順にソートできるようになりました。
* サマリビュー: 相関モード: 選択した指標に基づいてすべての列をソートし、その上で最初/最後の何列のみを表示できるようになりました。


* テーブルビュー: 表示する列の数を選択できるようになりました。


* データラングリング: 列の並べ替えのダイアログに特定の列を「先頭」および「最後」の動かすためのアイコンを追加しました。


* ダッシュボード: 「指定した高さに収める」が選択された場合、ある行の高さをドラッグ＆ドロップで調整した際、他の行の高さは維持されるようになりました。


* チャート: 箱ヒゲ図で、値に'inf'を含む場合、それらの値を自動で除外するようになりました。
* チャート: 繰り返しを選択したときに、チャートの上部に 「X軸を同期」オプションが表示されるようになりました。


* パラメーター: ステップデータを自動的にキャッシュすることにより、サーバーでインタラクティブモードを有効にする際のパフォーマンスが向上しています。
* パラメーター: 無効なパラメーターが使われている場合、より分かりやすいエラーメッセージが表示されます。


* プロジェクト: プロジェクトの作成日、ファイルサイズ、現在のメモリ消費量など、より多くの情報を確認できます。


* インストール: より直感的にインストールできるようにインストール時のUIをアップデートしました。


### 修正された問題

* データソース: 2,200列のCSVファイルをインポートしたときにデータのインポートが終了しない。
* データソース: マージされたCSV/Excelのデータソースにファイルを追加すると、最初のファイルのデータのみが表示される。


* サマリビュー: Date（日付） 型の列で、NAが正しくカウントされない。
* サマリビュー: 直前のステップで列名を変更すると画面が空白になる。


* データラングリング: 列名を変更した後にサマリビューに移動すると、「Another R process running」というエラーが表示される。
* データラングリング: フィルター: 年/月オプションが、日本語ロケールのWindowsで動作しない。
* データラングリング: 列名を変更すると、それ以降のステップが壊れる。
* データラングリング: マージ: 「データ・タイプを強制的にそろえる」はデフォルトでオフとなります。これはデフォルトのパフォーマンスを良くするためです。
* データラングリング: 空の文字列を他の文字列に置き換えると処理が終了しない。


* チャート: チャートに列を割り当て、「チャートデータをExcelでエクスポート」 オプションを選択した場合、「Another R process running」というエラーが表示される。
* チャート: 「一意な値のチェック」のダイアログで、「これ以降、この列では一意な値のチェックをスキップする」にチェックをつけても、他の列の割り当てを更新すると、「一意な値のチェック」ダイアログが表示される。
* チャート: レーダーチャートで、Character（文字）型の列を割り当て、プロパティから「値を標準化する」オプションを有効にすると、エラーが表示される。
* チャート: データが NaNを含む場合、バーチャートでソートが正しく動作しない。
* チャート: テーブルで、 "Inf"/"-Inf"が"NaN"と表示される。


* パラメーター: パラメータ名を変更すると、「Another R process running」というエラーが表示される。


* プロジェクト: プロジェクトを開こうとすると、「TypeError: Cannot read properties of null (reading 'classList') for some occasions.」と表示され、プロジェクトを開けない。
* プロジェクト: プロジェクトリストの画面でウィンドウサイズを変更すると、プロジェクトリストに不要なスペースが表示される。
* プロジェクト: エクスポートしたプロジェクトをインポートすると、ノートに埋め込まれた画像が含まれない。


* 一般: ネットワークエラーのダイアログの内容を分かりやすくしました。
* 一般: 列数の多いデータファイルをインポートする際のパフォーマンスを改善しました。
* 一般: 一部の環境で、Let's Encrypt SSL証明書のルートCA証明書の期限切れにより、exploratory.ioへのログインまたはOAuthのためのネットワーク接続が失敗することがありました。
* 一般: オフライン環境での起動するとき、Windowsのホームディレクトリの名前にマルチバイト文字を含む場合、「Cannot read properties of null (reading 'setDate')」というエラーが表示される。
